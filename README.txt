CPU and Memory Performance Monitor

This project uses Python to monitor and display system performance information such as CPU usage and memory usage. The program retrieves real-time system statistics and prints them to the console.

Language Used:
Python

Libraries Used:
psutil

Main Skills Practiced:
- system monitoring
- retrieving hardware statistics
- working with external Python libraries
- performance analysis
- formatted output

Project Features:
- Displays current CPU usage percentage
- Displays current memory usage percentage
- Uses real-time system performance data
- Monitors computer resource usage
- Demonstrates basic system analysis with Python
